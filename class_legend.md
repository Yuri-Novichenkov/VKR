# Легенда классов для визуализации

## Стандартная классификация LiDAR (LAS формат)

Согласно стандарту LAS, классы обычно означают:

- **Класс 0** - Never classified / Unclassified (Не классифицировано)
- **Класс 1** - Ground (Земля/поверхность)
- **Класс 2** - Low vegetation (Низкая растительность)
- **Класс 3** - Medium vegetation (Средняя растительность)
- **Класс 4** - High vegetation (Высокая растительность)
- **Класс 5** - Building (Здание)
- **Класс 6** - Low point / Noise (Низкая точка/шум)
- **Класс 7** - Model key-point (Ключевая точка модели)
- **Класс 8** - Water (Вода)
- **Класс 9** - Reserved (Зарезервировано)
- **Класс 10** - Reserved (Зарезервировано)
