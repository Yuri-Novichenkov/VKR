# Легенда классов для визуализации

## Стандартная классификация LiDAR (LAS формат)

Согласно стандарту LAS, классы обычно означают:

- **Класс 0** - Never classified / Unclassified (Не классифицировано)
- **Класс 1** - Ground (Земля/поверхность)
- **Класс 2** - Low vegetation (Низкая растительность)
- **Класс 3** - Medium vegetation (Средняя растительность)
- **Класс 4** - High vegetation (Высокая растительность)
- **Класс 5** - Building (Здание)
- **Класс 6** - Low point / Noise (Низкая точка/шум)
- **Класс 7** - Model key-point (Ключевая точка модели)
- **Класс 8** - Water (Вода)
- **Класс 9** - Reserved (Зарезервировано)
- **Класс 10** - Reserved (Зарезервировано)

## Интерпретация вашей визуализации

На основе структуры облака точек:

- **Темно-синий (Класс 0)** - Основание/пол, горизонтальные поверхности
- **Светло-синий (Класс 1)** - Вертикальные структуры (стены, объекты)
- **Оранжевый (Класс 2)** - Крупные вертикальные объекты (здания, мебель)
- **Остальные классы** - Другие объекты сцены

## Рекомендации для презентации

1. **Объясните структуру**: покажите, как модель разделила сцену на логические части
2. **Подчеркните качество**: модель корректно выделила горизонтальные (пол) и вертикальные (объекты) структуры
3. **Покажите применение**: такая сегментация полезна для навигации роботов, анализа архитектуры, создания 3D-моделей
